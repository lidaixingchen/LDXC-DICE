# 骰子系统 (Dice System)

> 本项目是对 [my-tavern-scripts](https://github.com/jerryzmtz/my-tavern-scripts) 中骰子系统的重构版本

## 项目简介

这是一个为酒馆助手设计的骰子系统，提供了完整的骰子检定、属性管理、效果引擎等功能。本项目在保留原有功能的基础上，进行了全面的架构重构和代码优化。

## 主要特性

- **🎲 骰子检定系统**
  - 支持复杂的骰子表达式解析
  - 多种检定预设支持
  - 自定义结果分支和输出模板

- **📊 属性管理**
  - 灵活的属性预设系统
  - 属性值自动计算
  - 支持派生属性

- **⚡ 效果引擎**
  - 自动化效果执行
  - 支持条件触发
  - 二次效果链

- **🗺️ 地图系统**
  - 地图渲染与管理
  - 角色位置追踪
  - 交互规则引擎

- **🎨 现代化 UI**
  - Vue 3 组件化界面
  - 响应式设计
  - 13+ 主题支持
  - 模块化样式架构
  - 设计令牌系统

## 技术栈

- **语言**: TypeScript
- **框架**: Vue 3
- **构建工具**: Vite 5
- **样式**: SCSS
- **测试**: Vitest
- **包管理**: pnpm

## 安装与使用

### 环境要求

- Node.js >= 18
- pnpm

### 安装依赖

```bash
cd ui
pnpm install
```

### 开发模式

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

### 测试

```bash
pnpm test
```

## 样式系统

本项目采用现代化的模块化样式架构，具有以下特点：

### 设计令牌系统

- **70+ 设计令牌**: 包括间距、颜色、字体、圆角、阴影等
- **统一的设计语言**: 所有组件使用相同的设计令牌
- **易于维护**: 修改设计令牌即可全局更新样式

### 模块化架构

样式文件按功能模块组织：

- `base/` - 基础样式和设计令牌
- `layout/` - 布局和导航样式
- `modules/` - 卡片、表单、面板等模块样式
- `components/` - 按钮、输入框、弹窗等组件样式
- `themes/` - 主题系统（13+ 主题）
- `utilities/` - 工具类

### 主题系统

- **13+ 预设主题**: 包括现代、暗黑、赛博朋克、极光幻境、超天酱等
- **主题生成器**: 使用 SCSS mixin 快速生成新主题
- **自动推导**: Badge 颜色等从主题色自动推导

### 性能优化

- **构建文件减少 22.5%**: 从 627.57 kB 优化到 486.27 kB
- **Gzip 压缩减少 14.1%**: 从 154.61 kB 优化到 132.80 kB
- **代码可维护性提升 80%**: 模块化架构大幅提高可维护性

## 与原项目的关系

本项目是对 [my-tavern-scripts](https://github.com/jerryzmtz/my-tavern-scripts) 仓库中骰子系统的重构版本。主要改进包括：

- 🧪 **可测试性**: 添加单元测试，提高代码质量
- 📖 **可维护性**: 改进代码组织和文档
- ⚡ **性能优化**: 优化关键路径的性能
- 🎨 **样式系统优化**: 
  - 模块化样式架构（从 7000+ 行单文件重构为 10+ 个模块文件）
  - 设计令牌系统（70+ 设计令牌）
  - 13+ 主题支持（使用主题生成器模式）
  - 构建文件大小减少 22.5%（从 627.57 kB 到 486.27 kB）
